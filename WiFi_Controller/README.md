# WiFi_Controller
esp32をアクセスポイントとしてスマホを接続します。WiFiUDP通信でコントローラを作成できます。
# アプリケーション
[WiFi TCP/UDP Controller](https://play.google.com/store/apps/details?id=udpcontroller.nomal&hl=ja)を使って、スマホでソフトウェアコントローラを作成できます。
